## 1. アプリの基本操作

### テキスト入力

画面中央のテキストエリアに文字を入力すると、以下の5項目がリアルタイムで表示されます：

| 項目 | 説明 |
|---|---|
| 文字数（スペース含む） | 全角・半角スペースを含むすべての文字数 |
| 文字数（スペース除く） | スペースを除いた文字数 |
| 単語数 | スペースで区切られた単語の数 |
| 行数 | 改行で区切られた行の数 |
| 段落数 | 空行で区切られた段落の数 |

### コピーボタン

テキストエリアの内容をクリップボードにコピーします。
コピー成功時は「コピーしました」とトースト通知が表示されます。

### クリアボタン

テキストエリアの内容をすべて消去します。

## 2. 文字数制限プリセット

テキストエリアの上にあるプリセットボタンを押すと、文字数制限モードになります。

### 用意されているプリセット

| プリセット | 文字数制限 |
|---|---|
| Twitter | 140文字 |
| Instagram キャプション | 2,200文字 |
| note 見出し | 100文字 |
| メタディスクリプション | 120文字 |
| ブログタイトル | 32文字 |
| カスタム | 任意の数値を入力可能 |

### プログレスバー

プリセット選択時に表示されます：

- **青**: 通常（0〜80%）
- **黄色**: 残り少ない（80%超）
- **赤**: 文字数超過（超過文字数も表示）

「なし」ボタンを押すと制限モードを解除します。

### カスタム設定

任意の文字数を入力欄に入力して「設定」ボタンを押すか、Enterキーで設定できます。

---

## 3. ダークモード

画面右上の丸いボタンで切り替えできます。

- 🌙（月アイコン）→ ダークモードに切替
- ☀️（太陽アイコン）→ ライトモードに切替

初回アクセス時は、お使いのPC/スマホのシステム設定に自動追従します。
手動で切り替えた場合、次回以降もその設定が記憶されます。

## 4. 名前のカスタマイズ

アプリのタイトルに「○○さんの文字カウンター」と表示する機能です。
3つの方法があります。

### 方法A：環境変数で固定する（配布・販売用）★推奨

Vercelの環境変数 `VITE_OWNER_NAME` に名前を設定します。
この場合、利用者は名前を変更できません。

→ 詳細は「5. 顧客への配布・販売方法」を参照

### 方法B：URLパラメータで設定する

```
https://あなたのURL.vercel.app/?name=田中太郎
```

URLに `?name=○○` をつけてアクセスすると名前が表示されます。
ブラウザに記憶されるので、次回以降はパラメータなしでも表示されます。

※ 環境変数が設定されていない場合のみ有効

### 方法C：アプリ画面から設定する

画面最下部の「名前を設定する」リンクから設定モーダルを開けます。

※ 環境変数が設定されていない場合のみ表示

## 5. 顧客への配布・販売方法

### 仕組み

```
GitHubリポジトリ（1つだけ）
     │
     │ 同じリポジトリから複数プロジェクトを作成
     │
     ├── Vercelプロジェクト① あなた用（テスト・開発）
     │     環境変数なし → 名前は自由に変更可能
     │     URL: char-counter-red.vercel.app
     │
     ├── Vercelプロジェクト② 田中さん用
     │     VITE_OWNER_NAME = 田中太郎
     │     URL: char-counter-tanaka.vercel.app
     │
     └── Vercelプロジェクト③ 鈴木さん用
           VITE_OWNER_NAME = 鈴木花子
           URL: char-counter-suzuki.vercel.app
```

### 新しい顧客を追加する手順（約3分）

#### ステップ1：Vercelで新規プロジェクト作成

1. https://vercel.com/new を開く
2. GitHubリポジトリ一覧から `char-counter` を選択して「Import」
3. **プロジェクト名** を変更する（例: `char-counter-tanaka`）

#### ステップ2：環境変数を設定

1. 画面の「Environment Variables」セクションを開く
2. 以下を入力：
   - **Key**: `VITE_OWNER_NAME`
   - **Value**: 顧客の名前（例: `田中太郎`）
3. 「Add」をクリック

#### ステップ3：デプロイ

1. 「Deploy」をクリック
2. 1〜2分で公開完了
3. 発行されたURL（例: `https://char-counter-tanaka.vercel.app`）を顧客に共有

### 顧客に表示される画面

- タイトル：「田中太郎さんの文字カウンター」（固定）
- 名前変更ボタン：非表示（変更不可）
- フッター：「© OFFICENN」
- 全機能（カウント・プリセット・ダークモード等）が利用可能

### 既存顧客の名前を変更する場合

1. Vercelダッシュボードで該当プロジェクトを開く
2. 「Settings」→「Environment Variables」
3. `VITE_OWNER_NAME` の値を変更して「Save」
4. 「Deployments」タブ → 最新デプロイの「...」→「Redeploy」

### 注意事項

- Vercel無料プランのプロジェクト数は無制限
- GitHubリポジトリは1つだけでOK
- コードを更新してPushすると、全顧客分が自動で最新版に更新される
- 顧客ごとに独立したURLが発行される

## 6. アプリの更新方法

### 日常の更新フロー

```
① Cursorでコードを修正
② ターミナルで以下を実行：
     git add -A
     git commit -m "変更内容の説明"
     git push
③ Vercelが自動検知 → 全プロジェクトが自動再デプロイ
④ 1〜2分で更新完了
```

### 特定の顧客だけ更新したくない場合

Vercelダッシュボードで該当プロジェクトの自動デプロイをオフにできます：
Settings → Git → Auto Deploy → 無効化
