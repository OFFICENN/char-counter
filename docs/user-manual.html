操作説明

テキスト入力と文字数表示

画面中央のテキストエリアに文字を入力すると、以下の5項目がリアルタイムで表示されます。

項目説明
文字数（スペース含む）全角・半角スペースを含むすべての文字数
文字数（スペース除く）スペースを除いた文字数
単語数スペースで区切られた単語の数
行数改行で区切られた行の数
段落数空行で区切られた段落の数

コピー・クリア

  • コピーボタン：テキストエリアの内容をクリップボードにコピーします。成功時は「コピーしました」と通知が表示されます。
  • クリアボタン：テキストエリアの内容をすべて消去します。

文字数制限プリセット

テキストエリアの上にあるプリセットボタンを押すと、文字数制限モードになります。

プリセット文字数制限
Twitter140文字
Instagram キャプション2,200文字
note 見出し100文字
メタディスクリプション120文字
ブログタイトル32文字
カスタム任意の数値を入力可能

プログレスバーの色

  • 青：通常（0〜80%）
  • 黄色：残り少ない（80%超）
  • 赤：文字数超過（超過文字数も表示）

「なし」ボタンを押すと制限モードを解除します。カスタム設定では任意の文字数を入力して「設定」ボタンまたはEnterキーで設定できます。

ダークモード

画面右上の丸いボタンで切り替えできます。

  • 🌙（月アイコン）→ ダークモードに切替
  • ☀️（太陽アイコン）→ ライトモードに切替

初回アクセス時はお使いの端末のシステム設定に自動追従します。手動で切り替えた場合、次回以降もその設定が記憶されます。
